# 04 — Modifiers and Operators

## Zweck

Dieses Dokument sammelt und definiert Operatoren und Modifier, die Zauber strukturieren oder verändern.

## Vorläufige Unterscheidung

| Typ | Bedeutung |
|---|---|
| Operator | Verknüpft oder steuert Wirkungen logisch. |
| Modifier | Verändert Parameter einer Wirkung. |

## Vorläufige Operatoren

| Operator | Beschreibung | Status |
|---|---|---|
| Sequenz | Wirkung A geschieht vor Wirkung B. | Hypothese |
| Parallelität | Wirkung A und B laufen gleichzeitig. | Hypothese |
| Bedingung | Wirkung tritt nur ein, wenn eine Bedingung erfüllt ist. | Hypothese |
| Wiederholung | Wirkung wird mehrfach oder periodisch angewendet. | Hypothese |
| Bindung | Wirkung bleibt an Ziel, Ort oder Zustand gekoppelt. | Hypothese |
| Umkehrung | Richtung oder Bedeutung einer Wirkung wird invertiert. | Hypothese |

## Vorläufige Modifier

| Modifier | Beschreibung | Betroffene Parameter |
|---|---|---|
| Verstärkung | Erhöht Intensität. | Energie, Risiko, Effektstärke |
| Abschwächung | Reduziert Intensität. | Energie, Präzision, Effektstärke |
| Dauer | Verlängert oder begrenzt zeitliche Wirkung. | Zeit, Kosten |
| Reichweite | Verändert räumliche Distanz. | Entfernung, Energie, Präzision |
| Form | Gibt räumliche Struktur vor. | Fläche, Volumen, Richtung |
| Zielpräzision | Schränkt Zielauswahl ein. | Fehlerquote, Kosten |
| Verzögerung | Wirkung tritt später ein. | Zeit, Bedingung |

## Offene Fragen

1. Sind Operatoren sichtbare Zeichen, mentale Strukturen oder beides?
2. Können Modifier auf Modifier angewendet werden?
3. Wie verhindert das System rekursive oder unendlich starke Kombinationen?
4. Gibt es inkompatible Kombinationen, z. B. `sofort` + `verzögert`?
