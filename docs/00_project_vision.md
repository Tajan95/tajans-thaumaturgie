# 00 — Project Vision

## Ziel

Tajan's Thaumaturgie soll ein granular ableitbares Magie-System entwickeln, das später digital simuliert werden kann.

Magie wird dabei nicht als Liste einzelner Zauber verstanden, sondern als formales System aus:

- Entitäten
- Zuständen
- Wirkungen
- Operatoren
- Modifiern
- Bedingungen
- Kosten
- Fehlerfällen

## Leitprinzip

> Ein Zauber darf nicht einfach existieren, sondern muss aus kleineren, nachvollziehbaren Komponenten logisch ableitbar sein.

## Zielbild

Ein Zauber soll langfristig wie eine Art Programm formulierbar sein:

```text
Ziel + Wirkung + Modifier + Bedingung + Kostenmodell → simulierbarer Effekt
```

## Abgrenzung

Dieses Projekt trennt bewusst zwischen:

| Ebene | Frage |
|---|---|
| Theorie | Was ist im System grundsätzlich wahr? |
| Regelmechanik | Wie wird ein Zauber konstruiert und begrenzt? |
| Simulation | Wie lässt sich die Regel technisch abbilden? |
| Gameplay | Wie wird daraus eine interessante Spielerfahrung? |
| Lore | Wie wird das System innerhalb einer Welt gedeutet? |
