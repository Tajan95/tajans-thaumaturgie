# 05 — Energy, Costs and Limits

## Zweck

Dieses Dokument sammelt Modelle dafür, was Magie begrenzt.

Ein gutes Magie-System braucht Grenzen, sonst wird jeder Zauber beliebig.

## Vorläufige Kostenfaktoren

| Faktor | Beschreibung | Status |
|---|---|---|
| Energiemenge | Aufwand für die Zustandsänderung selbst. | Offene Frage |
| Komplexität | Aufwand für präzise oder mehrteilige Struktur. | Hypothese |
| Reichweite | Größere Distanz erhöht Kosten oder Fehlerquote. | Hypothese |
| Dauer | Längere Wirkung erhöht Kosten oder Bindungsaufwand. | Hypothese |
| Zielwiderstand | Material, Masse oder Eigenstabilität des Ziels erschwert Veränderung. | Hypothese |
| Informationsmangel | Unklare Ziele oder Zustände erhöhen Risiko. | Hypothese |
| Konzentration | Mentale Kontrolle als begrenzender Faktor. | Offene Frage |
| Medium | Kanal oder Träger der Wirkung. | Offene Frage |

## Vorläufige Kostenformel

```text
Kosten = Basiswirkung × Masse/Fläche/Volumen × Intensität × Dauer × Komplexität × Distanzfaktor × Widerstand
```

**Status:** Hypothese

## Mögliche Fehlerfälle

| Fehlerfall | Ursache |
|---|---|
| Effekt verpufft | Zu wenig Energie oder falscher Zielbezug. |
| Effekt streut | Ungenaue Zielauswahl oder Form. |
| Effekt kehrt zurück | Fehlerhafte Bindung oder Umkehrung. |
| Effekt kollabiert | Interne Widersprüche in Syntax oder Semantik. |
| Nebeneffekt | Unberücksichtigte Material- oder Zustandsrelationen. |

## Offene Grundfragen

1. Ist Energie eine eigene Ressource oder eine Abstraktion für Veränderungsaufwand?
2. Gibt es Erhaltungssätze für Magie?
3. Muss jeder Zauber physikalisch kompatibel sein, oder darf Magie lokal Naturgesetze überschreiben?
4. Wie werden irreversible Veränderungen bepreist?
