# 02 — Magic Ontology

## Zweck

Dieses Dokument definiert, welche Arten von Dingen im Magie-System überhaupt existieren.

Die Ontologie beantwortet die Frage:

> Worauf kann Magie wirken, und welche Zustände oder Eigenschaften können verändert werden?

## Vorläufige Ontologie

| Kategorie | Beschreibung | Status |
|---|---|---|
| Entität | Ein unterscheidbares Ding im System, z. B. Objekt, Lebewesen, Fläche, Partikelwolke. | Hypothese |
| Zustand | Aktuelle Eigenschaften einer Entität, z. B. Position, Temperatur, Form, Bewegung. | Hypothese |
| Eigenschaft | Ein einzelner beschreibbarer Aspekt eines Zustands. | Hypothese |
| Relation | Beziehung zwischen Entitäten, z. B. Nähe, Bindung, Besitz, Ausrichtung. | Hypothese |
| Wirkung | Eine Veränderung an Zustand, Eigenschaft oder Relation. | Hypothese |
| Operator | Eine logische Operation, die Wirkungen strukturiert. | Begriff |
| Modifier | Eine Veränderung der Stärke, Richtung, Dauer, Form oder Bedingung einer Wirkung. | Begriff |
| Ziel | Das Objekt oder Gebiet, auf das ein Zauber angewendet wird. | Begriff |
| Medium | Der Träger oder Kanal, durch den eine Wirkung vermittelt wird. | Offene Frage |
| Energie | Ressource oder Aufwand, der magische Veränderung begrenzt. | Offene Frage |

## Offene Grundfrage

Muss Magie im System eine eigene Substanz/Energieform sein, oder ist sie eher eine Metasprache zur Veränderung vorhandener Zustände?

**Status:** Offene Frage

## Arbeitsannahme

Bis zur Klärung behandeln wir Magie als regelgebundene Zustandsmanipulation:

```text
Magie = Operation auf Entitäten, Zuständen, Eigenschaften und Relationen
```
