# 2D Prototype

Dieser Ordner ist für spätere technische Experimente zur 2D-Simulation vorgesehen.

## Mögliche erste Simulationsziele

- einfache Entitäten auf einem Grid oder in kontinuierlichem 2D-Raum
- Zustände wie Position, Temperatur, Bewegung, Materialtyp
- Wirkungen als Zustandsänderungen
- Zielauswahl über Koordinaten, Radius oder Kontakt
- Kostenberechnung für einfache Transformationen
- Fehlerfälle bei unklarer Zielauswahl oder zu hohen Kosten

## Status

Noch kein Prototyp implementiert.
